day01 Learning threejs

发表于 2018-12-28 | 分类于 three.js | 浏览

作为2019年的开始第一个月，本人开始学习three.js的章节。以《Learning Three.js》的中文版，作为学习书籍，这本书共有278页，平均每天下载9页的学习量，然后在此作为学习记录。

主要方式是：

  • 通过阅读书本，思考一些问题
  • 搜集一些概念，加以理解整理

初衷

先说说我为什么要学习three.js,如果说php是世界上最好的语言，那么JavaScript在我心中就是世界上无所不能的！事实好像亦是如此～咳咳…在浏览器中利用JavaScript创建一个三维的场景，这似乎是一件很酷的事情。对于一些很酷的事情，我比较感兴趣，况且之前的VR、AR如此火热，数据的可视化等，利用三维更加直观。坚信在不久的未来，JavaScript在web中的制作三维效果将会更加普遍；

现代浏览器开始支持webGL，通过webGL可以直接使用显卡资源来创建高新能的二维和三维图形，但是直接使用webGL编程来创建三维动画场景十分复杂而且容易出问题，使用threejs库可以简化这个问题；

什么是webGL？

webGL

WebGL is a cross-platform, royalty-free web standard for a low-level 3D graphics API based on OpenGL ES，exposed to ECMAScript via the HTML5 Canvas element.
最直白的解释就是webGL是一个web标准；他的特点就是跨平台、royalty-free（什么意思不明白），基于openGL ES的、用于创建低级三维模型的，然后通过Canvas元素暴露给ECMAScript里面的。
